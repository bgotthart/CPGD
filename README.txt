CPGD
====

XCode Version:
Da ein XCode Projekt nicht so einfach ohne 
Entwicklergebühr auf einem iPhone lauffähig zu machen ist, 
gibt es somit keine lauffähige Version des iOS Apps. 
Um das Projekt auszuführen, muss das XCode-Projekt 
(ArrowShooting3000_iOS > ArrowShooting3000_iOS.xcodeproj) gestartet, 
und in XCode auf Run geklickt werden. 
Der Pfad zum Ordner GameLogic und images sollte 
allerdings so beibehalten werden, da die C++ Files 
und Bilder als Referenz eingebunden sind. 

Android Version:
Im Ordner Runnable befindet sich ein AroowShooting3000_Android.apk, 
dass auf einem Android-Phone mit einer 
mindesten Android-Version 4 direkt auf das Phone zu speichern 
und dann ausführbar ist.

PC Version:
Im Ordner Runnable befinden sich ausführbare Dateien für
32bit-Windows Version:
arrowShooting3000_32bit.jar
arrowShooting3000_32bit.exe

64bit-Windows Version:
arrowShooting3000_64bit.jar
arrowShooting3000_64bit.exe
